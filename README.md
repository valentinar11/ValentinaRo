# UNOPS eTendering System - Mockup

Interactive HTML mockup for the UNOPS eTendering procurement platform.

## View the Mockup

Open `index.html` in your browser or visit the GitHub Pages URL.

## Features

- 7-tab navigation structure
- Event Setup, Timings, Scope, Submissions, Evaluation, Suppliers, Review
- Interactive forms with conditional fields
- Responsive design
- UNOPS branding

## Usage

This is a prototype/mockup for requirements gathering and design validation.
